1. Documentation of PhoneGap is available at http://docs.phonegap.com/phonegap-build/getting-started/
2. Notice that you have to edit file index.html variable PREFIX_URL to point to the location of the back-end in the cloud
3. Notice how the original app in node.js has been factorized to separate cleanly the client and server side
4. Go to: https://build.phonegap.com
5. Sign in and/or log in
6. Click on Apps
7. Select Update Code and upload there a .zip file with the contents of this folder
8. Download the APK generated
9. Transfer it through Bluetooth to your device
